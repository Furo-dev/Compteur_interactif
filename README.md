# Compteur Interactif

## Description
Une application web permettant d'incrémenter ou de décrémenter un compteur de manière interactive.

## Fonctionnalités
- Incrémentation du compteur.
- Décrémentation du compteur.
- Réinitialisation à zéro.

## Technologies utilisées
- HTML
- CSS
- JavaScript
